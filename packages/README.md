# Packages

This directory is reserved for officially maintained myx capability packages.  Over time, we plan to include a curated set of first‑party capabilities maintained by the myx project, such as wrappers for common APIs (GitHub, Slack, Postgres) and utilities (email, browser automation).

Packages in this directory will follow the package format described in `rfcs/0002-package-format.md` and may serve as canonical examples for the community.  Each package will live in its own subdirectory with its own `myx.yaml` and `capability.json`.

Currently there are no official packages; see `examples/` for early prototypes.
