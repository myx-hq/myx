# GitHub Capability

## Description

This is an example SKILL file generated from an myx package.  It exposes a simple command to search GitHub repositories.

## Commands

| Command | Description |
|--------|-------------|
| `search_repositories` | Search GitHub repositories by text query. |

## Usage

Use this skill when a user asks about repositories, issues or pull requests on GitHub.  The skill may require a `GITHUB_TOKEN` secret to access private repositories.
